# Indrajit Ghosh - Personal Portfolio

A modern, highly interactive, and visually stunning 3D personal portfolio website. Built to showcase projects, skills, and experience with a focus on rich aesthetics, smooth micro-interactions, and premium user experience.

## ✨ Features

- **3D Graphics & Environments**: Immersive 3D scenes powered by React Three Fiber and Three.js.
- **Smooth Animations**: High-performance, staggered, and scroll-triggered animations utilizing Framer Motion and GSAP.
- **Silky Smooth Scrolling**: Integrated Lenis for a premium, buttery-smooth scrolling experience.
- **Modern UI Design**: Beautiful typography, glassmorphism, responsive layouts, and curated color palettes structured via Tailwind CSS.
- **Type-Safe**: Developed completely in TypeScript for robust and maintainable code.

## 🛠️ Tech Stack

- **Core**: [React 19](https://react.dev/), [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) (`clsx`, `tailwind-merge`)
- **3D Rendering**: [Three.js](https://threejs.org/), [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber), [@react-three/drei](https://github.com/pmndrs/drei)
- **Animations**: [Framer Motion](https://www.framer.com/motion/), [GSAP](https://gsap.com/)
- **Scroll Hijacking**: [Lenis](https://lenis.darkroom.engineering/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/indrajit1306/IndrajitGhosh-Portfolio.git
   ```

2. Navigate into the directory:
   ```bash
   cd IndrajitGhosh-Portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

To start the local development server:

```bash
npm run dev
```

Your app will be running at `http://localhost:5173`.

### Production Build

To create a production-ready build:

```bash
npm run build
```

You can preview the built version using:

```bash
npm run preview
```
